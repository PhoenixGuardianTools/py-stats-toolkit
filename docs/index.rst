Installation
------------

You can install py-stats-toolkit using pip:

.. code-block:: bash

   pip install py-stats-toolkit

For development installation:

.. code-block:: bash

   git clone https://github.com/your-username/py-stats-toolkit.git
   cd py-stats-toolkit
   pip install -e ".[dev]"

Quick Start
----------

Here's a quick example of how to use py-stats-toolkit:

.. code-block:: python

   from py_stats_toolkit import YourMainClass

   # Create an instance
   instance = YourMainClass()

   # Use the instance
   result = instance.process_data()
